REMOTE INSTALLATION:

1) UNPACK THE PACKAGE (.ZIP FILE)
2) COPY VIA FTP AL FILE INSIDE THE "CMS" LOCAL DIRECTORY INTO YOUR HOSTING SERVICE

* IF THE WEB APPLICATION DON'T START, ADD "WRITE" PERMISSION TO "\App_Data" DIRECTORY (follow the instruction of your web hosting).

LOCAL INSTALLATION:

1) UNPACK THE PACKAGE (.ZIP FILE)
2) RUN "Visual Studio for Web" (*the express edition is free!), and select "File>Open Website" TO OPEN THE WEBSITE UNDER THE DIRECTORY "CMS"

*Download Visual Studio Express for Web
